# easy-mta

シンプルで使いやすい MTA（Mail Transfer Agent）ツールです。

## 概要

**easy-mta** は、メール送信処理を手軽に行うための軽量な MTA ツールです。複雑な設定を必要とせず、最小限のセットアップですぐに使い始めることができます。

## 特徴

- シンプルな設定ファイル
- SMTP によるメール送信
- 軽量・高速な動作
- 開発・テスト環境への素早い導入
